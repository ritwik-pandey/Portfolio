# 💻 Desktop OS-Themed Portfolio

A highly interactive, OS-inspired personal portfolio built to emulate a desktop environment. Experience a unique way to explore projects, skills, and background through draggable windows, a functional dock, and file system navigation.

## ✨ Features

- **Draggable & Resizable Windows:** Interact with the portfolio just like a real operating system.
- **Functional Dock:** Quick access to different sections seamlessly.
- **Apps & Utilities:**
  - **Finder/Portfolio:** Browse through projects like standard OS folders and files.
  - **Terminal/Skills:** View technical skills and proficiencies.
  - **Articles/Safari:** Check out my competitive programming profiles.
  - **Contact:** Get in touch via email or social links.
  - **About Me:** Learn about my background and interests.
- **Smooth Animations:** Powered by GSAP and Tailwind CSS for fluid, native-feeling transitions.
- **State Management:** Fully reactive window management utilizing Zustand.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS
- **Animations:** GSAP (GreenSock Animation Platform)
- **State Management:** Zustand
- **Icons:** Lucide React, Custom SVGs

## 📂 Featured Projects

- **PostNet:** A modern web platform designed to simplify how users create, manage, and share posts.
- **Let's Leet:** A Chrome extension to enhance LeetCode problem-solving with real-time insights.
- **Globe Quest:** An interactive web app designed to make exploring the world fun and educational.

## ⚙️ Local Development

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ritwik-pandey/portfolio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in your browser:**
   Navigate to `http://localhost:5173` to see the portfolio in action.
